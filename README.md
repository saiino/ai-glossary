# ai-glossary — AI用語 学習トラッカー

AI関連用語の理解度を記録する、個人用の学習トラッカーです。
Claude Code に「RAG理解した」のように話しかけると台帳(`terms.json`)が更新され、GitHub Pages のページに反映されます。
https://saiino.github.io/ai-glossary/

## 仕組み

```
「◯◯理解した」と発言
    ↓
Claude Code(vocab-tracker スキル)が terms.json を更新して commit・push
    ↓
GitHub Pages(index.html)に習熟度が反映される
```

## ファイル構成

| ファイル | 役割 |
|---|---|
| `terms.json` | 用語台帳。用語名・説明・関連語(related)・理解度(level)・更新日を持つ |
| `index.html` | 閲覧ページ。習熟度ヒートストリップ、レベル別フィルタ、検索、用語詳細モーダル |
| `.claude/skills/vocab-tracker/SKILL.md` | Claude Code 用スキル。会話から台帳を更新する手順書 |

## 理解度レベル

| level | 意味 |
|---|---|
| 0 | 未学習 |
| 1 | なんとなく |
| 2 | 理解した |
| 3 | 人に説明できる |

## 使い方(Claude Code での発言例)

- 「RAG理解した」 → 該当用語の level を更新
- 「ファインチューニングって何?」 → 説明してくれる(理解したら台帳更新)
- 「◯◯を追加して」 → 新しい用語を登録
- 「今どのくらい覚えた?」 → 学習状況のサマリー
- 「◯◯の説明もっとわかりやすくして」 → 会話で合意した表現に説明文を改善

## ページの見方

- **ヒートストリップ**: 1マス = 1用語。色が理解度。タップで詳細モーダルが開く
- **用語一覧**: 行をタップすると説明・関連語・更新日をモーダルで表示。関連語タグをタップすると用語間を回遊できる

## ローカルでの確認

`terms.json` を fetch で読み込むため、file:// では動きません。ローカルサーバー経由で開いてください。

```sh
npx http-server -p 8791 -c-1
# → http://localhost:8791
```
